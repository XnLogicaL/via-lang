# via

A performant, modern, minimal alternative to Lua.
via is a "sandbox" language that is aimed to be a modern alternative to Lua, it utilizes the C++23 standard to enhance the backend and uses modern conventions to fit in better with our current technology. It has ~1.5x more features compared to Lua, yet only a codebase the size of ~20k lines. Most features are kept as optional to preserve simplicity. via has a **massive** focus on compile-time analysis and runtime error prevention.
