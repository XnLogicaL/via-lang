\doxysubsection{Detailed Description}
Defines the representation of bytecode instructions in the Via VM. 

This file provides the core structures and type definitions used to represent low-\/level bytecode instructions that the virtual machine executes. It includes operand types, the main {\ttfamily Instruction} structure, and associated metadata for debugging.
