\doxysubsection{Detailed Description}
Represents a single VM instruction in the Via bytecode format. 

Each instruction has\+:
\begin{DoxyItemize}
\item An opcode ({\ttfamily op}) that specifies the operation to perform.
\item Up to three 16-\/bit operands ({\ttfamily a}, {\ttfamily b}, and {\ttfamily c}), whose semantics depend on the opcode.
\end{DoxyItemize}

The structure is aligned to 8 bytes to optimize for memory layout and access efficiency.
