\doxysubsection{Detailed Description}
A growable, dynamically sized array of {\ttfamily \doxylink{structvia_1_1Value}{Value}} elements. 

\begin{DoxySeeAlso}{See also}
\doxylink{structvia_1_1CSize}{CSize}
\end{DoxySeeAlso}
This structure wraps a heap-\/allocated buffer of {\ttfamily \doxylink{structvia_1_1Value}{Value}} entries and supports index-\/based access with automatic capacity expansion. Internally, resizing is delegated to the {\ttfamily \doxylink{structvia_1_1CSize}{CSize}} helper, which tracks the logical size and performs bounds checks.
