\doxysubsection{Detailed Description}
Represents a single function invocation\textquotesingle{}s execution context. 

Each {\ttfamily \doxylink{structvia_1_1CallFrame}{Call\+Frame}} holds\+:
\begin{DoxyItemize}
\item A pointer to the {\ttfamily \doxylink{structvia_1_1Closure}{Closure}} being executed (compiled function and captured upvalues).
\item A pointer to a local variables array ({\ttfamily locals}).
\item The size of the local variables array ({\ttfamily locals\+\_\+size}).
\item A saved program counter ({\ttfamily savedpc}) indicating where execution resumes after return.
\end{DoxyItemize}

Copy operations are disabled via {\ttfamily VIA\+\_\+\+NOCOPY}, but move semantics are allowed via {\ttfamily VIA\+\_\+\+IMPLMOVE}.
