\doxysubsection{Detailed Description}
A dynamically allocated hash table mapping {\ttfamily const char\texorpdfstring{$\ast$}{*}} keys to {\ttfamily \doxylink{structvia_1_1Value}{Value}} objects. 

This dictionary implementation is based on open addressing (linear probing). Keys are raw C strings assumed to be interned or otherwise stable.
