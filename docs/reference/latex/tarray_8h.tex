\doxysubsection{Detailed Description}
Declares the Array structure used for dynamic, indexed storage in the virtual machine. 

The Array structure implements a dynamic array of {\ttfamily Value}s with automatic resizing and index-\/based access. Arrays are core collection types in the via language runtime.
