\doxysubsection{Detailed Description}
Wraps a \doxylink{structvia_1_1Callable}{Callable} with its captured upvalues for lexical scoping. 

A \doxylink{structvia_1_1Closure}{Closure} is created when a function expression references non-\/local variables.
