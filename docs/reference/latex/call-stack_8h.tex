\doxysubsection{Detailed Description}
Defines the call stack and call frame structures for function execution. 

The call stack is a runtime structure that manages active function calls during the execution of programs. Each function call is represented by a {\ttfamily Call\+Frame}, which stores the execution context for that function, including its closure, local variables, and return address. The {\ttfamily Call\+Stack} holds a fixed number of frames, ensuring bounded recursion and stack safety.
