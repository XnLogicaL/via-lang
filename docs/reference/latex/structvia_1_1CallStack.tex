\doxysubsection{Detailed Description}
Represents the function call stack of the interpreter. 

Contains a fixed-\/size array of {\ttfamily \doxylink{structvia_1_1CallFrame}{Call\+Frame}} structures and a count of the current depth. Used to manage the execution of nested or recursive function calls.
