Detailed Description

Declares visitor interfaces for the Abstract Syntax Tree (AST) used in the via compiler.

This file provides a hierarchy of visitor classes that traverse or transform the AST of the via programming language. These include visitors for expression evaluation, type inference, and statement processing. The VisitorContext struct stores shared visitor state, including register allocation and error tracking.
